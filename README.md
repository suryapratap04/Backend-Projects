# Backend-Projects

### About Repository

This repository Contains the backend Projects

### Setting up Locally


1. First change the Directory

```bash
cd Folder-name
```

1. Copy the `.env.example` to `.env` and enter details to .env file

```bash
cp .env.example .env
```

2. Install Packages via npm

```bash
npm install
```

3. Run the Node.JS Development Server

```bash
node index.js
```
